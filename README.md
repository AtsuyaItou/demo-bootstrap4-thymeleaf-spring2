# Thymeleaf 3.0 with Spring Boot 2.0 web application

Development environment

* Java 1.8.0
* Spring Boot 2.0.2
* H2
* Maven 3.5.3

## Build & Run

```text
mvn clean package
```

```text
java -jar .\target\demo.jar
```

```text
http:localhost:9000/app/
```
