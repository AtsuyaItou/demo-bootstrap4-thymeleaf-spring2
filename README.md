# Thymeleaf 3.0 with Spring Boot 2.0.1 web application

## Build & Run

```text
mvn clean package
```

```text
java -jar .\target\demo-0.0.1-SNAPSHOT.jar
```

```text
http:localhost:9000/app/
```
